## Descripción del proyecto

Este repositorio contiene dos soluciones para un mismo problema utilizando diferentes paradigmas de programación:

- Programación Tradicional.
- Programación Orientada a Objetos.

Ambos programas permiten registrar y mostrar información básica de mascotas en Python
### Programación tradicional

La solución se desarrolla mediante funciones y variables. Los datos de la mascota son ingresados por teclado y posteriormente mostrados en pantalla sin utilizar clases ni objetos.

### Programación Orientada a Objetos

La solución utiliza una clase llamada Mascota, la cual contiene atributos y métodos para representar y gestionar la información de una mascota. Se crean objetos de la clase y se ejecutan sus métodos para mostrar información y emitir sonidos.


## Reflexión

La Programación Tradicional permite resolver problemas de forma sencilla mediante funciones y variables, siendo adecuada para programas pequeños.

La Programación Orientada a Objetos facilita la organización del código mediante clases y objetos, permitiendo representar entidades del mundo real de manera más estructurada y reutilizable.

Durante el desarrollo de esta actividad fue posible identificar las diferencias entre ambos enfoques y comprender cómo la POO mejora la escalabilidad y el mantenimiento de los programas.
